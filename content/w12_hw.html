第一題

1-1:解

解釋語法:

  1. 程式會顯示「請輸入你的名字：」。
  2. 你輸入名字後，程式會說「你好, [你的名字]」。

簡單來說，input() 就是讓程式可以問你問題並拿到你的答案。

# 問用戶名字
name = input("請輸入你的名字：")
# 顯示用戶的名字
print("你好, " + name)

3:解

解釋語法:

1.在programming_book設三個變數。

2.列印programming_book變數。

3.在設一個wanted_book的變數相當於input()的效果。

4.列印wanted_book變數。

5.會回傳變數如果有在programming_book裡的變數會顯示是我們有賣，反之是寫抱歉我們沒有賣。

programming_books = ["Learn python", "Python for all",  "Intro to python"]
print (programming_books)
wanted_book = input("Hi! What book would you like to buy?")
print (wanted_book)
if wanted_book in programming_books:
    print ("Yes, we sell it!")
else:
    print ("Sorry, we don't sell that book")

4.解

解釋語法:

1.在shopping_list設三個變數。

2.列印shopping_list變數。

3.再設一個new_item的變數相當於input()的效果。

4.毀回傳變數如果有再shopping_list裡會列出購物清單，反之不會。

5.阿item_to_remove是想移除物品。

6.如果移除物品是在清單會說移除清單，反之會寫不在清單。

shopping_list = ["carrots", "chocolate", "olives"]
print (shopping_list)
new_item = input("What else do I have to buy?")

if new_item in shopping_list:
    print (new_item + " is/are already in the shopping list")
    print (shopping_list) 
else:
    shopping_list.append(new_item)
    print (shopping_list)
    item_to_remove = input("What do I have to remove?")

if item_to_remove in shopping_list:
    shopping_list.remove(item_to_remove)
    print (shopping_list)
else:
    print (item_to_remove + " is/are not in the list")
    print (shopping_list)

第六題

6.解

cities = ["San Diego", "Prague", "Cape Town", "Tokyo", "Melbourne"]
print(cities)              # 印出整個列表
print(cities[1])           # 印出索引為1的元素 (Prague)
print(cities[1:4])         # 印出索引1到3的元素 ['Prague', 'Cape Town', 'Tokyo']
print(cities[1:4:2])       # 印出索引1到3之間，步長為2的元素 ['Prague', 'Tokyo']
print(cities[0:3])         # 印出索引0到2的元素 ['San Diego', 'Prague', 'Cape Town']
print(cities[:3])          # 印出索引0到2的元素，省略start相當於從0開始 ['San Diego', 'Prague', 'Cape Town']
print(cities[2:5])         # 印出索引2到4的元素 ['Cape Town', 'Tokyo', 'Melbourne']
print(cities[2:])          # 印出索引2到最後的所有元素 ['Cape Town', 'Tokyo', 'Melbourne']
print(cities[0:5:2])       # 印出索引0到4，步長為2的元素 ['San Diego', 'Cape Town', 'Melbourne']
print(cities[::2])         # 印出步長為2的元素 ['San Diego', 'Cape Town', 'Melbourne']
print(cities[4])           # 印出索引4的元素 (Melbourne)
print(cities[-1])          # 印出最後一個元素 (Melbourne)
print(cities[-4:-1])       # 印出倒數第4到倒數第2的元素 ['Prague', 'Cape Town', 'Tokyo']
print(cities[3:0:-1])      # 印出從索引3到索引1的元素，步長為-1 ['Tokyo', 'Cape Town', 'Prague']
print(cities[-2:-5:-1])    # 印出倒數第2到倒數第5的元素，步長為-1 ['Tokyo', 'Cape Town', 'Prague']
print(cities[::-1])        # 印出整個列表反轉 ['Melbourne', 'Tokyo', 'Cape Town', 'Prague', 'San Diego']
